# 🏠 Hostel Booking System
A full-stack hostel room booking web application built with Node.js, Express, and MongoDB.  
It allows users to browse rooms, book them, and manage reservations with authentication and admin features.

# Features
User registration and login system
Browse available hostel rooms
View detailed room information
Room booking functionality
Booking management system
Admin panel for managing rooms
Image upload using Cloudinary
Secure authentication with Passport.js
Session management with Express-Session
Flash messages for user feedback
  
# Tech Stack
Node.js
 Express.js
 MongoDB & Mongoose
 EJS (Templating Engine)
 Passport.js (Authentication)
 Bootstrap (Frontend UI)
 Cloudinary (Image storage)
 Multer (File upload)
 connect-mongo (Session store)
